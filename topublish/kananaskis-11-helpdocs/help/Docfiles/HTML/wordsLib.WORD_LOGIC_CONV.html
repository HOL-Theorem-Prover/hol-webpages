WORD_LOGIC_CONV : conv
STRUCTURE
wordsLib
SYNOPSIS
Conversion based on WORD_LOGIC_ss.
DESCRIPTION
The conversion WORD_LOGIC_CONV converts word logic expressions into a canonical form.
EXAMPLE
- WORD_LOGIC_CONV ``a && (b !! ~a !! c)``
