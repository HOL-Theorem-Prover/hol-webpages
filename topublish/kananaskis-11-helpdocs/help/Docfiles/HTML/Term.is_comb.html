is_comb : term -> bool
STRUCTURE
Term
SYNOPSIS
Tests a term to see if it is a combination (function application).
DESCRIPTION
If term M has the form f x, then is_comb M equals true. Otherwise, the result is false.
FAILURE
Never fails
