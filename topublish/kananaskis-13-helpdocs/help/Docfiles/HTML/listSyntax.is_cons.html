is_cons : (term -> bool)
STRUCTURE
listSyntax
SYNOPSIS
Tests a term to see if it is an application of CONS.
DESCRIPTION
is_cons returns true of a term representing a non-empty list. Otherwise it returns false.
FAILURE
Never fails.
