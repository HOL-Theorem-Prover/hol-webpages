gcdset :(num -> bool) -> num

Definitions

gcdset_def 
⊢ ∀s.
      gcdset s =
      if (s = ∅) ∨ (s = {0}) then 0
      else
        MAX_SET
          ({n | n ≤ MIN_SET (s DELETE 0)} ∩ {d | ∀e. e ∈ s ⇒ divides d e})
