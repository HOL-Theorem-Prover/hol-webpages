A : ('a -> 'b) -> 'a -> 'b
STRUCTURE
Lib
SYNOPSIS
Combinator for function application
DESCRIPTION
The application A f x equals f x.
FAILURE
A f never fails. A f x fails if f x fails.
EXAMPLE
- map2 A [I, K 3, fn x => x + 1]  [1,2,3];

> val it = [1, 3, 4] : int list
