is_const : term -> bool
STRUCTURE
Term
SYNOPSIS
Tests a term to see if it is a constant.
DESCRIPTION
If c is an instance of a previously declared HOL constant, then is_const c returns true; otherwise the result is false.
FAILURE
Never fails.
