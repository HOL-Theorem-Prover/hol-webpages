is_pvar : (term -> bool)
STRUCTURE
pairSyntax
LIBRARY
pair
SYNOPSIS
Tests a term to see if it is a paired structure of variables.
DESCRIPTION
is_pvar "pvar" returns true iff pvar is a paired structure of variables. For example, ((a:*,b:*),(d:*,e:*)) is a paired structure of variables, (1,2) is not.
FAILURE
Never fails.
