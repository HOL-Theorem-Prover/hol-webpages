Definitions

fixed_width_def 
|- !n c. fixed_width n c <=> !x. domain c x ==> (LENGTH (encoder c x) = n)

|- !l n. l IN of_length n <=> (LENGTH l = n)

Theorems

fixed_width_univ 
|- !phi c n.
     wf_coder c /\ fixed_width n c ==>
     ((!x. domain c x ==> phi x) <=> !w::of_length n. phi (decoder c w))

of_length_univ_zero
