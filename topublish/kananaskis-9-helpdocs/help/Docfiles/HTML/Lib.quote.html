quote : string -> string
STRUCTURE
Lib
SYNOPSIS
Put quotation marks around a string.
DESCRIPTION
An application quote s is equal to "\"" ^ s ^ "\"". This is often useful when printing messages.
FAILURE
Never fails
EXAMPLE
- print "foo\n"; 
foo
> val it = () : unit

- print (quote "foo" ^ "\n");
"foo"
> val it = () : unit
