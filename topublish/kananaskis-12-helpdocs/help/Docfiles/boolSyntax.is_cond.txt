----------------------------------------------------------------------
is_cond                                                   (boolSyntax)
----------------------------------------------------------------------
is_cond : term -> bool

SYNOPSIS
Tests a term to see if it is a conditional.

DESCRIBE
If {M} has the form {if t then t1 else t2} then {is_cond M}
returns {true} If the term is not a conditional the
result is {false}.

FAILURE
Never fails.

SEEALSO
boolSyntax.mk_cond, boolSyntax.dest_cond.
