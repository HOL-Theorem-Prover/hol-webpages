append                                                           (Lib)
----------------------------------------------------------------------
append : 'a list -> 'a list -> 'a list

KEYWORDS
Curry

SYNOPSIS
Curried form of list append.

DESCRIBE
The function {append} is a curried form of the standard operation for
appending two ML lists.

FAILURE
Never fails.

EXAMPLE

- append [1] [2,3] = [1] @ [2,3];
> val it = true : bool
