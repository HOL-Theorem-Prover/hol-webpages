fail                                                        (Feedback)
----------------------------------------------------------------------
fail : unit -> 'a

SYNOPSIS
Raise a {HOL_ERR}.

KEYWORDS
exception

DESCRIBE
The function {fail} raises a {HOL_ERR} with default values. This is
useful when detailed error tracking is not necessary.

FAILURE
Always fails.

EXAMPLE

- fail() handle e => Raise e;

Exception raised at ??.??:
fail
! Uncaught exception:
! HOL_ERR
