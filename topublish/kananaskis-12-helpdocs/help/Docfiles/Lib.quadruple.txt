----------------------------------------------------------------------
quadruple                                                        (Lib)
----------------------------------------------------------------------
quadruple : 'a -> 'b -> 'c -> 'd -> 'a * 'b * 'c * 'd

SYNOPSIS
Makes four values into a quadruple.

DESCRIBE
{quadruple x1 x2 x3 x4} returns {(x1, x2, x3, x4)}.

FAILURE
Never fails.

SEEALSO
Lib.quadruple_of_list, Lib.pair, Lib.triple.
