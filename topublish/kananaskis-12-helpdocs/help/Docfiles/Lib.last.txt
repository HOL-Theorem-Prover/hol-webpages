----------------------------------------------------------------------
last                                                             (Lib)
----------------------------------------------------------------------
last : 'a list -> 'a

SYNOPSIS
Computes the last element of a list.

DESCRIBE
{last [x1,...,xn]} returns {xn}.

FAILURE
Fails if the list is empty.

SEEALSO
Lib.butlast, Lib.el, Lib.front_last.
