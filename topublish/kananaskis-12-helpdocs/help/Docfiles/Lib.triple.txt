----------------------------------------------------------------------
triple                                                           (Lib)
----------------------------------------------------------------------
triple : 'a -> 'b -> 'c -> 'a * 'b * 'c

SYNOPSIS
Makes three values into a triple.

DESCRIBE
{triple x y z} returns {(x, y, z)}.

FAILURE
Never fails.

SEEALSO
Lib.triple_of_list, Lib.pair, Lib.quadruple.
