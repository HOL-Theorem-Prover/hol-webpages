is_exists : term -> bool
STRUCTURE
boolSyntax
SYNOPSIS
Tests a term to see if it is an existential quantification.
DESCRIPTION
If M has the form ?v. t then is_exists M returns true. If the term is not an existential quantification the result is false.
FAILURE
Never fails.

mk_exists, dest_exists
