is_pforall : (term -> bool)
STRUCTURE
pairSyntax
LIBRARY
pair
SYNOPSIS
Tests a term to see if it is a paired universal quantification.
DESCRIPTION
is_pforall "!pair. t" returns true. If the term is not a paired universal quantification the result is false.
FAILURE
Never fails.
