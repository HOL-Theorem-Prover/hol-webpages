strcat : string -> string -> string
STRUCTURE
Lib
SYNOPSIS
Concatenates two ML strings.
FAILURE
Never fails.
EXAMPLE
- strcat "1" "";
> val it = "1" : string

- strcat "hello" "world";
> val it = "helloworld" : string

- strcat "hello" (strcat " " "world");
> val it = "hello world" : string
