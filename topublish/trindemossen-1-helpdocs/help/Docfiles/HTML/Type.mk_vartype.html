mk_vartype : string -> hol_type
STRUCTURE
Type
SYNOPSIS
Constructs a type variable of the given name.
FAILURE
Fails if the string does not begin with '.
EXAMPLE
- mk_vartype "'giraffe";
> val it = `:'giraffe` : hol_type

- try mk_vartype "test";

Exception raised at Type.mk_vartype:
incorrect syntax
