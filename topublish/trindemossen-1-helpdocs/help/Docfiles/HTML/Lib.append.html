append : 'a list -> 'a list -> 'a list
STRUCTURE
Lib
SYNOPSIS
Curried form of list append.
DESCRIPTION
The function append is a curried form of the standard operation for appending two ML lists.
FAILURE
Never fails.
EXAMPLE
- append [1] [2,3] = [1] @ [2,3];
