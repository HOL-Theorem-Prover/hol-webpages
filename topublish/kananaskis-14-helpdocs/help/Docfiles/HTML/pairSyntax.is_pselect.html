is_pselect : (term -> bool)
STRUCTURE
pairSyntax
LIBRARY
pair
SYNOPSIS
Tests a term to see if it is a paired choice-term.
DESCRIPTION
is_select "@pair. t" returns true. If the term is not a paired choice-term the result is false.
FAILURE
Never fails.

is_select, dest_pselect
